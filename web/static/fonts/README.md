# Local Fonts

This directory is reserved for redistributable local font files.

Recommended families:

- Inter for Latin text
- Vazirmatn for Persian, Arabic, and mixed RTL/LTR text
- JetBrains Mono for monospace technical values

No binary font files are committed until their exact source and license are verified. The application CSS uses those family names first, then falls back to platform fonts so development never depends on remote font loading.
